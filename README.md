# Quick-PAY Project

## 개발환경
- platform : M1 chip(ARMv8-A)
- Language : Java17(temurin)
- SpringBoot : 3.2.1
- Database : MySQL 8.0.35
- NoSQL : Redis 7.2.3

## Redis/MySQL 설치
- 디렉토리 이동 /your/project/root/quick-pay
```shell 
docker compose up -d
```

## Redis/MySQL 종료
- 디렉토리 이동 /your/project/root/quick-pay
```shell
docker compose down
```
